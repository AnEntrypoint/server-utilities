# @sequential/server-utilities

Server utilities including cache management, task executor, logging, and configuration helpers.

## Installation

```bash
npm install @sequential/server-utilities
```

## Usage

```javascript
import {{ main }} from '@sequential/server-utilities';

const result = main(/* args */);
```
